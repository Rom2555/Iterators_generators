# Итераторы и генераторы: плоское представление вложенных списков

Python-скрипт, демонстрирующий работу с итераторами и генераторами для преобразования вложенных списков в одномерные («выпрямление» списков).

## Функционал

- `FlatIterator` — итератор для списков второго уровня вложенности.
- `flat_generator` — генератор для тех же списков.
- `flat_recursive_generator` — рекурсивный генератор, обрабатывающий списки любой глубины.

## Тесты

- `test_1()` — проверка `FlatIterator`.
- `test_2()` — проверка `flat_generator`.
- `test_3()` — проверка рекурсивного разворачивания.

## Запуск
`bash python main.py`


Подходит для изучения протоколов итераторов и генераторов в Python.
